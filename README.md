## 下载

### Docker

```
docker pull apachecn0/uiuc-cs241-notes-zh
docker run -tid -p <port>:80 apachecn0/uiuc-cs241-notes-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g uiuc-csersiyi-notes-zh
uiuc-csersiyi-notes-zh <port>
# 访问 http://localhost:{port} 查看文档
```
